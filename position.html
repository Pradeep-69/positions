Python C JavaScript

Python is a high-level, interpreted programming language known for its simplicity and readability. It's widely used for web development, data analysis, artificial intelligence, automation, and more.

C is a general-purpose, procedural programming language developed in the early 1970s. It's known for speed and control, widely used in system and embedded programming.

JavaScript is used to make web pages interactive. It powers buttons, animations, scrolling effects, and more.
